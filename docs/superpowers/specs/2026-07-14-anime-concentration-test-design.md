# 二次元浓度测试 - 设计文档

## 项目概述

### 背景
为参加AI创造力大赛而开发的二次元浓度测试网页应用，旨在通过多种题型全面测试用户的二次元知识水平，参考B站答题模式，打造权威且有趣的测试体验。

### 目标
- 创建一个视觉冲击力强、充满二次元风格的测试网页
- 涵盖近10年热门番剧、游戏、声优、梗文化等广泛领域
- 提供多样化题型（听力、图片、选择、填空）
- 生成详细的测试结果报告和等级称号
- 支持社交分享功能

### 参考设计风格
- B站（Bilibili）：粉色主题、弹幕文化、沉浸式体验
- Crunchyroll：渐变色彩、卡片式布局
- 日系动漫游戏UI：霓虹光效、粒子特效、动态交互

---

## 技术架构

### 技术栈
- **前端**：HTML5 + CSS3 + JavaScript（纯前端单页应用）
- **图标**：Emoji + 自定义SVG
- **字体**：Google Fonts（Noto Sans SC）
- **存储**：LocalStorage（保存答题记录）

### 文件结构
```
anime-test.html          # 主页面（包含所有HTML/CSS/JS）
```

### 核心模块
1. **欢迎页模块**：介绍测试、开始按钮
2. **答题模块**：题目展示、选项交互、进度管理
3. **计时模块**：倒计时功能
4. **评分模块**：计算分数、生成结果
5. **分享模块**：生成分享卡片

## UI设计规范

### 色彩方案

| 颜色用途 | 色值 | 说明 |
|---------|------|------|
| 主色调 | #fb7299 | B站粉色，热情活力 |
| 辅助色1 | #7C3AED | 深紫色，神秘创意 |
| 辅助色2 | #06B6D4 | 青蓝色，清新科技 |
| 辅助色3 | #F97316 | 橙色，活力热情 |
| 背景色 | #0F0F1A | 深色背景，沉浸式体验 |
| 卡片背景 | #1A1A2E | 层次分明 |
| 文字主色 | #FFFFFF | 白色 |
| 文字次要 | #A1A1AA | 浅灰色 |
| 霓虹光效 | rgba(0, 212, 255, 0.5) | 发光效果 |

### 视觉元素
- **渐变背景**：粉紫渐变模拟二次元梦幻感
- **霓虹光效**：按钮、卡片边框添加发光效果
- **粒子特效**：背景飘浮星星/花瓣
- **动漫风格图标**：自定义手绘风格图标

### 字体规范
- **标题**：圆润手绘风格，大号字体
- **正文**：思源黑体，确保可读性
- **日文元素**：适当加入假名或短句增强氛围

### 布局特点
- **卡片式设计**：圆角卡片，悬浮阴影效果
- **响应式布局**：适配手机和桌面端
- **流畅动画**：页面切换、答题反馈添加过渡动画

## 功能实现

### 核心功能

1. **欢迎页**
   - 展示测试介绍
   - 开始测试按钮
   - 参与人数统计

2. **答题系统**
   - 题目随机抽取（从题库中随机选择）
   - 多种题型支持
   - 进度条显示
   - 倒计时功能

3. **音频播放**
   - 支持HTML5音频播放
   - 播放/暂停/重播控制

4. **图片展示**
   - 角色图片、场景截图
   - 图片缩放支持

5. **结果生成**
   - 分数计算
   - 等级称号生成
   - 雷达图绘制
   - 答题详情

6. **社交分享**
   - 生成分享卡片
   - 复制结果链接

7. **数据持久化**
   - LocalStorage保存答题记录
   - 支持重新测试
