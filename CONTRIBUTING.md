# 参与贡献

感谢你改进二次元浓度研究所。

## 本地开发

```bash
npm install
npm run dev
```

提交改动前请运行：

```bash
npm run check
```

## 题库贡献

- 题目 ID 必须唯一，且每题恰好包含四个非空选项。
- `answer` 使用 0–3 的数组索引。
- 请附上简洁、可核验的答案说明，避免主观争议题。
- 图片必须记录来源和版权归属；无法确认再分发边界时，优先使用原创 SVG 或不使用图片。
- 不要提交个人数据、访问令牌、账号凭据或未授权的付费内容。

## Pull Request

请说明改动动机、影响页面和验证方式。涉及 UI 的改动建议附桌面端与移动端截图；涉及计分或抽题逻辑的改动必须补充测试。
