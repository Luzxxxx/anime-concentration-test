## 二、核心功能

### 1. 三种挑战模式
- **轻松模式**：10题，适合快速测试
- **标准模式**：15题，均衡测试体验
- **困难模式**：20题，深度挑战

### 2. 双计分系统
- **知识掌握度**：百分制，基于基础分加权计算
- **竞技表现**：分数/上限，包含速度分和连击分

### 3. 能力雷达
- 五维能力评估：番剧、游戏、声优、梗文化、综合常识
- 小样本可信度规则：0题显示"未覆盖"、1-2题显示"样本不足"、≥3题显示百分比

### 4. 角色匹配
- 根据答题表现匹配二次元角色
- 显示匹配度和维度分析

### 5. 成就系统
- 40+ 成就徽章
- 稀有度分级：普通、稀有、史诗

### 6. 社交分享
- 生成通行证海报
- 生成二维码
- 复制成绩链接

## 九、运行方式

### 本地运行

```bash
# Python
python -m http.server 8080

# Node.js
node server.js
```

访问 http://localhost:8080/index.html 即可体验。

### 离线运行

本应用支持离线运行，只需将 `index.html` 和 `assets/` 目录放在本地服务器环境中即可。
